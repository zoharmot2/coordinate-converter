Coordinate Converter

Convert a single coordinate or an Excel/CSV file using a searchable local CRS catalog, including WGS 84, Israeli grids, UTM, Web Mercator, British National Grid & Lambert-93.

This tool converts coordinates between common geographic and projected coordinate reference systems. You can work with one point at a time, process an entire spreadsheet, inspect results on an interactive map, and copy or share converted coordinates.

How to use the converter

Single coordinate

  1. Search for and select the source CRS.
  2. Search for and select the target CRS.
  3. Enter X/Y or Longitude/Latitude values.
  4. Select UTM zone and hemisphere when required.
  5. Click Convert coordinates.

Excel / CSV batch

  1. Open the Excel / CSV batch tab.
  2. Upload an XLSX, XLS or CSV file.
  3. Select the worksheet, header row, X column and Y column.
  4. Select the source and target CRS.
  5. Convert and download the resulting XLSX file.

Interactive map

  1. Click the map to choose a WGS 84 location.
  2. Use the map point as a new input.
  3. Inspect coordinates under the cursor in several CRS formats.
  4. Look up the nearest mapped place when needed.

Supported CRS catalog

Search is local and works without an API key. Version 2.4 also supports shareable conversion links that preserve the selected systems and coordinate.

Validation: Numeric format, coordinate order, CRS bounds, area of use, UTM zone and hemisphere, and transformation consistency are checked before results are accepted.
Accuracy notice: Suitable for general GIS and educational use. Cadastral or surveying work should be verified with an authoritative geodetic service.
